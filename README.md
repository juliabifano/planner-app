# 📋 Planner App

Aplicação web de planejamento diário desenvolvida com React, com foco em organização de tarefas por data e experiência do usuário.

## 🚀 Funcionalidades
- Adição de tarefas por dia
- Edição e remoção de tarefas
- Marcação de tarefas como concluídas
- Organização com horário (opcional)
- Interface responsiva

## ✨ Animações e Experiência do Usuário

O projeto utiliza a biblioteca Framer Motion para criar uma experiência mais fluida e moderna:

✔️ Entrada suave das tarefas (fade + slide)
✔️ Remoção animada de tarefas
✔️ Reorganização automática com animação (layout animation)
✔️ Microinterações (hover e clique)
✔️ Feedback visual ao concluir tarefas

## 🛠️ Tecnologias
- React
- JavaScript
- HTML5
- CSS3 / Tailwind
- Framer Motion

## 💡 Objetivo
Projeto desenvolvido para praticar React e evoluir na transição para a área de tecnologia.

📦 Instalação
npm install
npm run dev
